# 0x0E. C - Function Pointers
---
## Description

This project in the Lower Level Programming series is about:
* What are function pointers and how to use them
* What does a function pointer exactly hold
* Where does a function pointer point to in the virtual memory

---
File|Task
---|---
0-print_name.c | Prints a name
1-array_iterator.c | Executes a function given as a parameter on each element
2-int_index.c | Searches for an integer
3-main.c | Main function for the calculator
3-op_functions.c | Contains all the calculation formulas
3-get_op_func.c | Determines the operator
3-calc.h | Contains struct and prototypes
